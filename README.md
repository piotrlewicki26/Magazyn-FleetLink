# Magazyn-FleetLink

Magazyn urządzeń do lokalizacji pojazdów + oferty

## Struktura projektu

| Plik | Opis |
|------|------|
| `index.html` | Strona główna – osadza generator ofert w ramce (iframe) |
| `FleetLink_Generator_Ofert_umowa_18_3.html` | FleetLink Generator Ofert GPS – pełny system tworzenia ofert i umów |

## Jak uruchomić

1. Otwórz `index.html` w przeglądarce – generator ofert zostanie automatycznie załadowany.
2. Alternatywnie otwórz bezpośrednio `FleetLink_Generator_Ofert_umowa_18_3.html`.

## Funkcje generatora ofert

- 🏠 **Strona tytułowa** – dane firmy, klienta, numer oferty
- 🏢 **O nas** – prezentacja firmy i usług
- 📋 **Pozycje oferty** – tabela produktów/usług z cenami, rabatami i VAT
- 📝 **Umowa** – szablon umowy z warunkami (OWU)
- 📱 **Urządzenia** – zarządzanie inwentarzem urządzeń GPS z importem CSV/Excel
- 📄 **Eksport PDF / DOCX** – generowanie dokumentów ofertowych i umów
